# Changelog

## [0.2.5] - 2024-12-29

### Added
- **Scroll View Mode** - Toggle between paginated A4 view and continuous scroll view
- **Export to HTML** - Export documents as standalone HTML files with folio pagination
- **Loading indicator** when changing themes

### Fixed
- EPUB files can now be opened from Recent Files
- PDF export no longer fails on large documents (fixed data URL length limit)

### Changed
- Updated foliojs to 0.2.2

## [0.2.4] - 2024-12-28

### Added
- EPUB support with full HTML cleanup
- PlantUML and Mermaid diagram rendering
- Export to PDF and DOCX

### Fixed
- PlantUML encoding issues
- Empty heading handling in TOC
- Scroll listener and TOC navigation with async content

## [0.2.3] - 2024-12-27

### Added
- Multi-tab support
- Recent files list
- Style templates (11 themes)

## [0.2.2] - 2024-12-26

### Added
- Table of Contents with page numbers
- Keyboard navigation
- Zoom controls

## [0.2.1] - 2024-12-25

### Added
- LaTeX/KaTeX support
- Basic markdown rendering

## [0.2.0] - 2024-12-24

### Added
- Initial release with paginated A4 view
- Electron desktop app
